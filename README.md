## 🏥 What is DiagnosTenAI?

</div>

> **DiagnosTenAI** is an intelligent, AI-powered healthcare web application
> that predicts the risk of **10 different diseases** using advanced
> Machine Learning algorithms trained on real-world medical datasets.
> Users can enter their medical information, receive instant AI-driven
> risk assessments, and download professional PDF health reports —
> all within a beautifully designed, responsive web interface.

- **10 independently trained** Machine Learning models
- Each model optimized for its specific medical condition
- Real-time predictions with confidence percentage scoring
- Clear risk level classification (High Risk / Low Risk)

### 📝 Smart Dynamic Forms
- Auto-generated input forms based on selected disease
- Intelligent field types: numerical, dropdown, categorical
- User-friendly labels replacing complex medical codes
- Input validation and error handling built-in

### 📄 Professional PDF Reports
- Downloadable medical-grade report per prediction
- Includes patient info, inputs, result, confidence & timestamp
- AI-generated health analysis summary
- Clean, professional healthcare formatting

### 🎨 Beautiful Responsive UI
- Modern healthcare-themed interface design
- Fully responsive across all screen sizes and devices
- Smooth Dark Mode / Light Mode toggle

## ⚙️ Installation & Setup

### Prerequisites
Make sure you have the following installed:
- Python 3.8 or higher
- pip package manager
- Git

STEP 1 ──► Open DiagnosTenAI in your browser
              │
STEP 2 ──► Select a disease from the dashboard
              │
STEP 3 ──► Fill in your medical information
           in the smart dynamic form
              │
STEP 4 ──► Click "Predict" to run AI analysis
              │
STEP 5 ──► View your prediction result:
           ✅ Risk Level
           📊 Confidence Score
           📝 AI Health Summary
              │
STEP 6 ──► Download your PDF Medical Report
